# ChattingRoom

A simple and easy online chatting room built by JavaScript,Socket.io, MaterializeCss and ExpressJS, deployed on Heroku.


You can get to the app by:
https://tragically-hockey-91723.herokuapp.com/



## Getting Started
```
npm install 
```

```
node server.js
```
